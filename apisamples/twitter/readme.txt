Follow the instructions on mytweets.py to set up a app as a developer on a twitter account and get the consumer key, secret, and oauth token and secret.

mytweets goes through recent tweets.

you can also explore with twitterUserTopWords.py that displays the top ten words from the most recent 200 tweets of a given user.

For example, getting CNN's top words from the command line would look like:
$ python twitterUserTopWords.py -u CNN

For example, getting Horse_ebooks's top words from the command line would look like:
$ python twitterUserTopWords.py -u Horse_ebooks

Barack Obama is the default user if you don't specify one from the command line:
$ python twitterUserTopWords.py
